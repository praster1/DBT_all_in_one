# DBT_all_in_one

> `DBT All In One` 원고와 Companion Pack을 GitHub에서 바로 읽기 좋은 Markdown 구조로 정리한 저장소입니다.

## 저장소 구조

| 경로 | 역할 |
| --- | --- |
| `chapters/` | 챕터/부록 단위 Markdown 원고 |
| `book_from_pdf.md` | 전체 책을 한 파일로 이어 읽을 수 있는 통합 Markdown |
| `00_meta/book_charter.md` | 책의 범위, 독자, 예제, 플랫폼 범위 |
| `01_outline/master_toc.md` | 파트/챕터 구조를 빠르게 보는 목차 |
| `07_styles/style_system.md` | GitHub용 Markdown 스타일 규칙 |
| `codes/` | 실행 가능한 Companion Pack |

## 권장 읽기 순서

1. [book_charter](./00_meta/book_charter.md)
2. [master_toc](./01_outline/master_toc.md)
3. [chapters/00_intro_and_toc.md](./chapters/00_intro_and_toc.md)
4. 필요한 챕터와 [codes/README.md](./codes/README.md)를 병행
